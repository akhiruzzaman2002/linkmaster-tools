# LinkMaster - লিংক ম্যানেজমেন্ট টুলস

একটি সম্পূর্ণ ফ্রি ওপেন-সোর্স লিংক ম্যানেজমেন্ট প্ল্যাটফর্ম।

## ফিচারস

- ✅ URL শর্টেনিং
- ✅ QR কোড জেনারেশন  
- ✅ বায়ো লিংক ক্রিয়েশন
- ✅ লিংক ট্র্যাকিং
- ✅ PWA সাপোর্ট
- ✅ অফলাইন কাজ
- ✅ বাংলা ভাষা সাপোর্ট

## ব্যবহার

1. `index.html` ফাইলটি ব্রাউজারে ওপেন করুন
2. বিভিন্ন টুলস ব্যবহার করুন
3. অফলাইনেও কাজ করবে
